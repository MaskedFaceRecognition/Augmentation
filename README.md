## 20210218 update
### 얼굴 인식과 GAN

#### 과제 방향 
1. 마스크 쓴 사람의 얼굴을 인식하고 (얼굴 인식 모델)
2. 그 사람의 index로 train set을 정해주고 
3. 그 train set에 GAN을 적용하여 마스크를 없앤 사진을 만든다.
→ 데이터 셋 만드는 것이 중요. 우선적으로 팀원 개개인의 사진을 이용하여 데이터 셋을 구축

#### 진행 방향
1. 데이터 셋 구축
2. 얼굴 인식 모델
3. GAN 모델
